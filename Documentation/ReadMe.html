Installation and Building

It must be built for use at your site.
  1. Copy orientalMotorIOC folder to your home directory
  2. Edit the configure/RELEASE.local file and set the paths to your installation of EPICS base, EPICSv4, ASYN support modules .
  3. Build StreamDevice (or add your streamdevice support module location to configure/RELEASE.local file):
    cd orientalMotorIOC/StreamDevice
    make
  4. Execute make in the top level directory.


ARD-CD Controller Hardware Configuration

  1. Set termination resistor setting switches (TERM. No1 and No2) ON
  2. Set SW4 Switch 1(Set Address Number) to Off, SW4 Switch 2(Set The Protocol) to On(Modbus)
  3. Set SW2(Transmission rate setting switch) to 0(9600 Baudrate)
    4. 0 - 9600
    1 - 19200
    2 - 38400
    3 - 57600
    4 - 115200
  4. Set Adress number setting switch (ID) to 1
    5. Change id in subs. file
    * OrientalMotor/orientalMotorIOC/orientalApp/template.substitutions
    file db/orientalMotor.db {
        pattern { sys, sec, dev, id }
        { "BL", "C001", "ORIENTAL", "0x01"} # ID = 0x01
    }


Note :
If "$(sys)-$(sec)-$(dev):POSITION" is set to 0, motor moves to $(sys)-$(sec)-$(dev):OFFPOSITION
If "$(sys)-$(sec)-$(dev):POSITION" is set to 1, motor moves to $(sys)-$(sec)-$(dev):ONPOSITION
$(sys)-$(sec)-$(dev):STATUS for monitoring POSITION state(OFF, ON or Between)
